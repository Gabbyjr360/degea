# She Steps Backend

Minimal Express + SQLite backend for product management.

## Endpoints
- `GET /products` — List all products
- `POST /products` — Add a product (JSON: name, price, image, desc)

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start server:
   ```bash
   npm start
   ```

## Database
- SQLite file: `products.db` (auto-created)
